# SketchUp Solid Inspector 2 for Linux Wine

## Installation

1. **Download the RBZ file** from the releases section
2. **Open SketchUp** and go to **Extensions > Extension Manager**
3. Click the **gear icon** (⚙️) in the Extension Manager window
4. **Enable "Unrestricted"** by checking the box if it's not already marked
5. Click the **Install Extension** button and select the downloaded `.rbz` file
6. **Restart SketchUp** to activate the extension

## Usage

Once installed, the solid inspector 2 will be available in SketchUp's menu system.
